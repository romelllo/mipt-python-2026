## Содержание

На этом семинаре мы изучим безопасность и тестирование:

- Аутентификация: OAuth2, JWT токены
- Авторизация и права доступа
- CORS (Cross-Origin Resource Sharing)
- Защита от распространённых уязвимостей
- Тестирование с pytest
- TestClient для интеграционных тестов
- Mocking и fixtures
