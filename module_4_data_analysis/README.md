# Модуль 4: Анализ и визуализация данных

## Описание

Четвертый модуль курса посвящен анализу и визуализации данных с использованием популярных Python библиотек.

## Темы

### 1. NumPy - Numerical Python
- Массивы NumPy (ndarray)
- Создание и манипуляция массивами
- Индексирование и срезы
- Математические операции
- Broadcasting
- Линейная алгебра
- Случайные числа

### 2. Pandas - Data Analysis
- Series и DataFrame
- Чтение и запись данных (CSV, Excel, SQL, JSON)
- Индексирование и выборка данных
- Группировка и агрегация (groupby)
- Объединение данных (merge, join, concat)
- Работа с пропущенными значениями
- Преобразование данных
- Time series анализ

### 3. Matplotlib - Визуализация
- Основы Matplotlib
- Figure и Axes
- Типы графиков:
  - Line plots
  - Scatter plots
  - Bar charts
  - Histograms
  - Pie charts
- Настройка графиков (цвета, стили, метки)
- Subplots и компоновка
- Сохранение графиков

### 4. Seaborn - Статистическая визуализация
- Основы Seaborn
- Темы и стили
- Статистические графики:
  - Distribution plots
  - Categorical plots
  - Regression plots
  - Matrix plots (heatmaps)
- Facet grids
- Pair plots

### 5. Plotly - Интерактивная визуализация
- Plotly Express
- Интерактивные графики
- Dashboards с Plotly Dash
- 3D визуализация
- Географические карты

### 6. Jupyter Notebooks
- Установка и использование Jupyter
- Markdown в notebooks
- Magic commands
- Расширения Jupyter
- JupyterLab vs Jupyter Notebook
- Экспорт notebooks

### 7. Основы анализа данных
- Exploratory Data Analysis (EDA)
- Описательная статистика
- Корреляционный анализ
- Обработка выбросов
- Feature engineering (введение)
- Работа с категориальными данными

### 8. Статистический анализ
- Основы статистики
- Распределения
- Гипотезы и тестирование
- SciPy для статистического анализа
- A/B тестирование
