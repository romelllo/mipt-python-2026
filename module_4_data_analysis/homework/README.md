# Домашние задания - Модуль 4

Домашние задания по анализу и визуализации данных.

## Список заданий

### Домашнее задание 1: Exploratory Data Analysis (EDA)
**Файл:** `homework_01_eda.ipynb`

Проведите полный исследовательский анализ данных выбранного датасета.

**Датасет:** Выберите один из предложенных или найдите свой на Kaggle

**Задачи:**
1. Загрузить и изучить данные
2. Описательная статистика
3. Обработка пропущенных значений
4. Обнаружение выбросов
5. Корреляционный анализ
6. Создать минимум 10 информативных визуализаций
7. Сформулировать выводы

**Требования:**
- Jupyter Notebook с markdown комментариями
- Использование pandas, numpy
- Минимум 3 типа графиков (scatter, hist, box, etc.)
- Интерпретация результатов

### Домашнее задание 2: Визуализация данных
**Файл:** `homework_02_visualization.ipynb`

Создайте комплексную визуализацию данных для выбранного датасета.

**Задачи:**
1. Создать информативный дашборд (dashboard)
2. Использовать разные типы визуализаций:
   - Временные ряды
   - Категориальные данные
   - Распределения
   - Корреляции
3. Интерактивные графики (Plotly)
4. Красивое оформление (Seaborn styles)

**Требования:**
- Использовать matplotlib, seaborn, plotly
- Минимум 8 различных графиков
- Единый стиль оформления
- Подписи и легенды на русском языке
- Выводы по каждой визуализации

### Домашнее задание 3: Анализ временных рядов
**Файл:** `homework_03_time_series.ipynb`

Проанализируйте данные временных рядов (цены акций, погода, продажи и т.д.).

**Задачи:**
1. Загрузить данные временного ряда
2. Визуализация тренда
3. Сезонная декомпозиция
4. Скользящие средние
5. Базовое прогнозирование
6. Визуализация результатов

**Требования:**
- Работа с datetime в pandas
- Resample и группировка по времени
- Визуализация трендов
- Интерпретация сезонности

### Домашнее задание 4: Проект по выбору
**Директория:** `homework_04_project/`

Самостоятельный проект по анализу данных.

**Этапы:**
1. Выбор датасета и темы
2. Постановка вопросов для анализа
3. Сбор/подготовка данных
4. Анализ и визуализация
5. Выводы и презентация

**Требования:**
- Jupyter Notebook с полным циклом анализа
- Минимум 15 визуализаций
- Презентация результатов (PDF или slides)
- Оформленный README с описанием проекта
