# Домашние задания - Модуль 1

Эта директория содержит домашние задания для самостоятельного выполнения.

## Общие правила

1. Каждое домашнее задание должно быть выполнено индивидуально
2. Код должен быть чистым и хорошо задокументированным
3. Следуйте PEP 8 стандарту оформления кода
4. Добавляйте комментарии для сложных участков кода
5. Тестируйте свой код перед отправкой

## Список домашних заданий

### Домашнее задание 1: Калькулятор
**Файл:** `homework_01_calculator.py`

Создайте простой калькулятор, который умеет выполнять базовые операции.

**Требования:**
- Поддержка операций: +, -, *, /, **
- Обработка ошибок (деление на ноль)
- Работа с вещественными числами

### Домашнее задание 2: Анализ текста
**Файл:** `homework_02_text_analysis.py`

Напишите программу для анализа текстового файла.

**Требования:**
- Подсчет количества слов
- Подсчет количества символов
- Нахождение самого длинного слова
- Частота встречаемости слов

### Домашнее задание 3: Система управления задачами
**Файл:** `homework_03_todo_list.py`

Создайте консольное приложение для управления списком задач.

**Требования:**
- Добавление задач
- Удаление задач
- Отметка задач как выполненных
- Сохранение задач в файл
- Загрузка задач из файла

## Сроки сдачи

Сроки будут объявлены преподавателем на занятии.

## Критерии оценки

- **Корректность** (40%) - программа работает правильно
- **Читаемость кода** (30%) - код легко понять
- **Обработка ошибок** (20%) - программа устойчива к ошибкам
- **Документация** (10%) - наличие комментариев и docstrings
