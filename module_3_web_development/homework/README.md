# Домашние задания - Модуль 3

Домашние задания по веб-разработке и сбору данных.

## Список заданий

### Домашнее задание 1: REST API на FastAPI
**Директория:** `homework_01_fastapi_blog/`

Создайте REST API для блога используя FastAPI.

**Требования:**
- CRUD операции для постов
- Система пользователей и аутентификация
- Комментарии к постам
- Теги/категории
- Пагинация
- Документация API (Swagger)

**Модели:**
- User (id, username, email, password_hash)
- Post (id, title, content, author_id, created_at)
- Comment (id, content, post_id, author_id, created_at)
- Tag (id, name)

**Эндпоинты:**
- POST /register - Регистрация
- POST /login - Авторизация
- GET /posts - Список постов
- POST /posts - Создать пост
- GET /posts/{id} - Получить пост
- PUT /posts/{id} - Обновить пост
- DELETE /posts/{id} - Удалить пост
- POST /posts/{id}/comments - Добавить комментарий

### Домашнее задание 2: Django приложение
**Директория:** `homework_02_django_ecommerce/`

Создайте интернет-магазин на Django.

**Требования:**
- Каталог товаров
- Корзина покупок
- Система заказов
- Админ-панель Django
- Аутентификация пользователей
- Django REST Framework для API

**Модели:**
- Product (название, описание, цена, изображение, категория)
- Category (название, описание)
- Cart (пользователь, товары)
- Order (пользователь, товары, статус, дата)

**Функциональность:**
- Просмотр каталога
- Добавление в корзину
- Оформление заказа
- История заказов
- Управление через админ-панель

### Домашнее задание 3: Веб-скрапинг
**Файл:** `homework_03_web_scraper/`

Разработайте веб-скрапер для сбора данных.

**Задачи:**
1. Собрать данные с новостного сайта
2. Сохранить в базу данных
3. Создать API для доступа к данным
4. Экспорт в CSV/JSON

**Требования:**
- Использовать BeautifulSoup или Scrapy
- Обработка пагинации
- Обработка ошибок
- Rate limiting
- Соблюдение robots.txt

**Данные для сбора:**
- Заголовки новостей
- Дата публикации
- Автор
- Текст новости
- Ссылка на источник
