# Jupyter Notebooks - Примеры

Эта директория содержит Jupyter notebooks с примерами анализа и визуализации данных.

## Список notebooks

1. **01_numpy_basics.ipynb** - Основы NumPy
   - Создание массивов
   - Индексирование и срезы
   - Математические операции
   - Broadcasting

2. **02_pandas_intro.ipynb** - Введение в Pandas
   - Series и DataFrame
   - Чтение данных
   - Фильтрация и группировка
   - Обработка пропусков

3. **03_data_visualization.ipynb** - Визуализация данных
   - Matplotlib basics
   - Seaborn статистические графики
   - Plotly интерактивные графики

4. **04_data_analysis_example.ipynb** - Полный пример анализа
   - Exploratory Data Analysis
   - Очистка данных
   - Визуализация
   - Выводы

## Запуск notebooks

```bash
# Установить Jupyter
pip install jupyter jupyterlab

# Запустить Jupyter Notebook
jupyter notebook

# Или JupyterLab (рекомендуется)
jupyter lab
```

## Датасеты

Примеры используют публичные датасеты:
- Titanic dataset
- Iris dataset
- Sales data

Все датасеты будут загружены автоматически при запуске notebooks.
