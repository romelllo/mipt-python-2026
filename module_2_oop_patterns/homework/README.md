# Домашние задания - Модуль 2

Домашние задания по объектно-ориентированному программированию и паттернам проектирования.

## Список заданий

### Домашнее задание 1: Система управления библиотекой
**Файл:** `homework_01_library_system.py`

Разработайте систему управления библиотекой с использованием ООП.

**Требования:**
- Классы: `Book`, `Author`, `Library`, `Member`
- Наследование для разных типов книг
- Инкапсуляция для защиты данных
- Методы для выдачи/возврата книг
- Обработка исключений

**Функциональность:**
- Добавление/удаление книг
- Регистрация читателей
- Выдача книг
- Возврат книг
- Поиск книг
- Отчеты

### Домашнее задание 2: Реализация паттернов
**Файл:** `homework_02_design_patterns.py`

Реализуйте несколько паттернов проектирования для решения конкретных задач.

**Требуется реализовать:**
1. **Factory Pattern** - для создания разных типов уведомлений (Email, SMS, Push)
2. **Observer Pattern** - для системы подписки на события
3. **Strategy Pattern** - для разных стратегий оплаты (Credit Card, PayPal, Crypto)

**Требования:**
- Корректная реализация паттернов
- Демонстрация использования
- Комментарии к коду

### Домашнее задание 3: SOLID принципы
**Файл:** `homework_03_solid_refactoring.py`

В файле предоставлен плохо написанный код. Ваша задача - провести рефакторинг с применением SOLID принципов.

**Задачи:**
- Применить Single Responsibility Principle
- Применить Open/Closed Principle
- Применить Liskov Substitution Principle
- Применить Interface Segregation Principle
- Применить Dependency Inversion Principle

**Критерии:**
- Код должен быть более читаемым
- Классы должны иметь одну ответственность
- Код должен быть легко расширяемым

## Критерии оценки

- **Корректность реализации ООП** (30%)
- **Применение паттернов** (30%)
- **Качество кода и архитектура** (25%)
- **Документация и комментарии** (15%)
