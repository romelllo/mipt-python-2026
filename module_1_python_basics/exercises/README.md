# Упражнения для Модуля 1

Эта директория содержит практические упражнения для отработки навыков программирования на Python.

## Структура упражнений

Каждое упражнение содержит:
- Описание задачи
- Примеры входных и выходных данных
- Тесты для проверки решения

## Список упражнений

1. **exercise_01_variables.py** - Работа с переменными
2. **exercise_02_strings.py** - Операции со строками
3. **exercise_03_lists.py** - Работа со списками
4. **exercise_04_dicts.py** - Словари
5. **exercise_05_functions.py** - Создание функций

## Как выполнять упражнения

1. Откройте файл с упражнением
2. Прочитайте описание задачи
3. Напишите решение в обозначенном месте
4. Запустите тесты для проверки

```bash
python exercise_01_variables.py
```

## Рекомендации

- Начинайте с простых упражнений
- Не смотрите на решение сразу - попробуйте сами
- Тестируйте разные варианты входных данных
- Задавайте вопросы, если что-то непонятно
