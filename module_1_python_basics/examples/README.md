# Примеры кода для Модуля 1

Эта директория содержит примеры кода для демонстрации основных концепций Python.

## Список примеров

1. `01_basic_syntax.py` - Базовый синтаксис Python
2. `02_data_structures.py` - Работа со структурами данных
3. `03_functions.py` - Функции и их использование
4. `04_file_operations.py` - Работа с файлами
5. `05_exceptions.py` - Обработка исключений

## Использование

Каждый файл можно запустить независимо:

```bash
python 01_basic_syntax.py
```

## Рекомендации

- Изучайте примеры последовательно
- Экспериментируйте с кодом
- Изменяйте примеры для лучшего понимания
