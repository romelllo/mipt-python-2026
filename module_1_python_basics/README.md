# Модуль 1: Настройка среды и основы Python

## Описание

Первый модуль курса посвящен базовым концепциям Python и настройке рабочей среды разработки.

## Темы

### 1. Настройка среды разработки
- Установка Python
- Выбор IDE (VS Code, PyCharm, Jupyter)
- Виртуальные окружения (venv, virtualenv)
- Менеджеры пакетов (pip, poetry, uv)
- Git и GitHub basics

### 2. Основы синтаксиса Python
- Переменные и типы данных
- Операторы
- Условные конструкции (if/elif/else)
- Циклы (for, while)
- Comprehensions (list, dict, set)

### 3. Структуры данных
- Списки (list)
- Кортежи (tuple)
- Словари (dict)
- Множества (set)
- Строки (str)

### 4. Функции
- Определение функций
- Параметры и аргументы
- Возвращаемые значения
- Lambda функции
- Декораторы (введение)

### 5. Модули и пакеты
- Импорт модулей
- Создание собственных модулей
- Пакеты Python
- `__init__.py`
- Стандартная библиотека Python

### 6. Работа с файлами
- Чтение и запись файлов
- Контекстные менеджеры (with)
- Работа с путями (pathlib)
- JSON, CSV форматы

### 7. Обработка исключений
- Try/except блоки
- Типы исключений
- Создание собственных исключений
- Finally и else в обработке исключений

## Структура модуля

В директории `exercises/` вы найдете упражнения для отработки навыков:
- Базовый синтаксис и операции
- Работа со структурами данных
- Написание функций
- Работа с файлами

## Домашние задания

В директории `homework/` размещаются задания для самостоятельной работы с подробными инструкциями.
