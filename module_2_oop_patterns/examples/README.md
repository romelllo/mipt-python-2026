# Примеры паттернов проектирования

Эта директория содержит примеры реализации основных паттернов проектирования на Python.

## Структура

### Порождающие паттерны (Creational)
- `singleton.py` - Singleton pattern
- `factory.py` - Factory Method pattern
- `builder.py` - Builder pattern

### Структурные паттерны (Structural)
- `adapter.py` - Adapter pattern
- `decorator.py` - Decorator pattern
- `facade.py` - Facade pattern

### Поведенческие паттерны (Behavioral)
- `strategy.py` - Strategy pattern
- `observer.py` - Observer pattern
- `command.py` - Command pattern

## Использование

Каждый файл содержит:
1. Описание паттерна
2. Пример реализации
3. Демонстрацию использования

Запускайте примеры:
```bash
python singleton.py
```
